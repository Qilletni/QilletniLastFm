# qilletni/lastfm

This is the Last.fm service provider for Qilletni. This allows you to use Last.fm in Qilletni, albiet relatively limited due to Last.fm's data-oriented/statistics model, versus playback.

Package docs: [https://docs.qilletni.dev/library/lastfm/](https://docs.qilletni.dev/library/lastfm/)
